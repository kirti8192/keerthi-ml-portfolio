# keerthi-ml-portfolio

- My portfolio of machine learning projects with an emphasis on deep learning, sequence modeling, and modern transformer architectures across NLP and time-series tasks.
- The idea is to have clean, reproducible PyTorch implementations and interpretability tools (attention heatmaps, gradient saliency, logit trajectories, hidden-state visualizations) to accompany each project, with the goal of learning by doing and building real hands-on experience.
